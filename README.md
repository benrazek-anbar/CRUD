# CRUD
Une application CRUD responsive développée avec HTML, Tailwind CSS et JavaScript.
Ce projet permet aux utilisateurs de :

Créer de nouveaux produits.

Afficher et gérer les données existantes.

Mettre à jour les informations des produits.

Supprimer un ou plusieurs éléments.

Rechercher par titre ou par catégorie.

Calculer automatiquement les prix et les totaux.

Ce projet est conçu pour améliorer les compétences en manipulation du DOM, en conception d'interfaces responsives et en gestion des données via localStorage.
